Application de récapitulatif journalier pour orthoptiste libéral.

Permet de saisir chaque jour les honoraires AMY et les actes RNO sur deux sites (Yssingeaux et Le Puy), avec calcul automatique des kilomètres parcourus. Un récapitulatif annuel est disponible et peut être imprimé en PDF.

Les données sont sauvegardées sur le serveur dans une base SQLite. L'utilisateur administrateur peut modifier les données ; les autres utilisateurs YunoHost accèdent en lecture seule.
